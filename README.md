## Description

This project is a simple **AI-powered interactive quiz application** that analyzes user answers and generates a short personality-style summary.

The main goal of the project is to demonstrate:
- handling step-based user interaction
- managing application state across multiple questions
- integrating AI for text generation based on structured input

<br>

## Quiz Flow

- User starts the quiz by typing **"start"**
- The first question is displayed
- User selects one of the answers: **A, B or C**
- After each answer, the next question appears automatically
- The process repeats until **7 questions** are completed
  <br>
  <br>

## User Interaction

- Simple text-based input system
- Controlled flow: no skipping questions
- Input validation (only A / B / C accepted)
- Clear progression through quiz steps
  <br>
  <br>

## AI Summary Generation

- After completing all questions, user responses are collected
- AI analyzes the pattern of answers
- A **personalized summary** is generated describing traits, tendencies or behavior patterns
- Output is displayed at the end of the quiz
  <br>
  <br>

## State Handling

- Tracks current question index
- Stores all user answers in order
- Controls quiz start and completion state
- Prevents restarting in the middle of the flow
  <br>
  <br>

## Description:

The project is **educational / experimental**.

It demonstrates how simple user input flow can be combined with AI-generated output to create an interactive experience.
